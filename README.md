# Lab Assignment LAW

## Prequisites

- Go
- NPM/Yarn

## Run Backend

- `cd backend`
- `go install`
- `go run main.go`

## Run Frontend

- `cd frontend`
- `npm install`
- `npm start`

### Link example Backend (REST): http://localhost:8000/api/sum

### Link example Frontend (Web): http://localhost:3000/?a=17&b=18
